# AI工具集网站

这是一个使用 Next.js 和 TypeScript 构建的 AI 工具导航网站。

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18

## 开始使用

1. 安装依赖：

```bash
npm install
```

2. 运行开发服务器：

```bash
npm run dev
```

3. 打开 [http://localhost:3000](http://localhost:3000) 查看网站。

## 项目结构

```
src/
  ├── app/              # Next.js 13+ App Router
  ├── components/       # React 组件
  ├── styles/          # 样式文件
  └── types/           # TypeScript 类型定义
```

## 构建生产版本

```bash
npm run build
```

## 运行生产版本

```bash
npm start
```
